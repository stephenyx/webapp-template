# Serena Memories

This directory contains memory files for Serena MCP integration.

## Available Memories

| File | Description |
|------|-------------|
| `architecture.md` | Tech stack and project structure overview |
| `conventions.md` | Coding standards and file naming conventions |
| `error-patterns.md` | Common errors and their solutions |

## Usage

Serena will automatically read these memories to:
- Understand project architecture
- Follow established conventions
- Troubleshoot common errors

## Adding New Memories

Create new `.md` files in this directory for:
- New architectural decisions
- Learned error patterns
- Project-specific conventions
